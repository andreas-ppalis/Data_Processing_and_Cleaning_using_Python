# Data Processing and Cleaning using Python

This project includes basic concepts of data processing and data cleaning using Python language. It is a worked example with random data that was created to show some of my skills in data preparation using Python. The dataset is referred to a soft drink product mentioning some of its descriptions that can be helpful for further analysis.

What to expect:
- Importing of useful libraries
- Basic information on tha dataset and some statistics
- How can we deal with missing values
- Find outliers of the dataset
